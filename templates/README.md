# Шаблоны документов

Все шаблоны используют Jinja-теги в синтаксисе [docxtpl](https://docxtpl.readthedocs.io/). Чтобы вставить тег в документ Word: вписать прямо в текст `{{ имя_поля }}` без форматирования (иначе Word разобьёт тег на runs и подстановка не сработает — лучше всего набирать тег вручную в одну операцию).

## application.docx — Заявление на приём

| Поле | Значение |
|---|---|
| `{{ full_name }}` | ФИО кандидата (как ввёл при регистрации) |
| `{{ phone }}` | Телефон кандидата в формате `+79001234567` |
| `{{ goal_title }}` | Название цели: «Трудоустройство», «Практика», «Стажировка» |
| `{{ application_id }}` | Внутренний ID заявки |
| `{{ date }}` | Дата формирования (ДД.ММ.ГГГГ) |

## medical_referral.docx — Направление на медосмотр

Те же поля, что в `application.docx`.

| Поле | Значение |
|---|---|
| `{{ full_name }}` | ФИО кандидата |
| `{{ phone }}` | Телефон |
| `{{ goal_title }}` | Цель |
| `{{ application_id }}` | ID заявки |
| `{{ date }}` | Дата формирования |

## practice_characteristic.docx — Характеристика практики/стажировки

| Поле | Значение |
|---|---|
| `{{ full_name }}` | ФИО кандидата |
| `{{ goal_title }}` | «Практика» / «Стажировка» |
| `{{ supervisor }}` | ФИО руководителя практики (вводит HR) |
| `{{ topic }}` | Тема практики/стажировки (вводит HR) |
| `{{ period_from }}` | Дата начала (ДД.ММ.ГГГГ) |
| `{{ period_to }}` | Дата окончания (ДД.ММ.ГГГГ) |
| `{{ date }}` | Дата формирования |

## Как создать шаблон

1. Откройте новый документ в MS Word.
2. Сверстайте «бланк»: шапка организации, заголовок, реквизиты, основной текст с пропусками.
3. В местах, где нужны подстановки, наберите `{{ имя_поля }}` точно как в таблице выше — без жирного, курсива и пробелов внутри скобок.
4. Сохраните как `имя.docx` (НЕ `.doc`, НЕ `.docm`) в эту директорию.
5. Перезапускать бота не нужно — шаблон подхватится при следующей генерации.

## Проверка шаблона

Локально протестировать рендер:

```python
from docxtpl import DocxTemplate
doc = DocxTemplate("templates/application.docx")
doc.render({
    "full_name": "Иванов Иван Иванович",
    "phone": "+79001234567",
    "goal_title": "Трудоустройство",
    "application_id": 1,
    "date": "18.04.2026",
})
doc.save("/tmp/test.docx")
```

Если в результате теги `{{ ... }}` остались — значит, Word разбил их на несколько runs. Удалите тег, переключитесь на чистый шрифт и наберите заново одним движением.

## Куда сохраняются результаты

Сгенерированные файлы складываются в `storage/<user_id>/<application_id>/generated/<kind>_<timestamp>.docx` и фиксируются в таблице `generated_documents`. HR может скачать их через карточку заявки → «📝 Сгенерированные».
